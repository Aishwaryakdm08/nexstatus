# NexStatus

Cloud-Based API Monitoring and Status Management Platform

## Features

- API Health Monitoring
- Incident Management
- Uptime Analytics
- Email Alerts
- Public Status Page
- AWS Deployment
- Infrastructure as Code using Terraform
